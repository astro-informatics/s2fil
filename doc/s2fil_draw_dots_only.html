Draw dots at specified positions on constant value map.

Usage:

  • [-help]: Display usage information.
  • [-nside nside]: Healpix nside of map to produce.
  • [-out filename_out]: Name of map HEALPix fits file to write. Map contains dots drawn at specified positions.
  • [-dots filename_dots]: Name of text file containing dot alpha and beta positions.
  • [-large large (optional)]: Logical to indicate whether to draw large dots.
